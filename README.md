## Features (by file)

- File-Encrypter/Decrypter: this script will encrypt/decrypt a file of your choosing from the directory you're running the script.
  - When decrypting you can choose an output name for the decrypted file.
  - Future expansion of the script will include choosing what encryption scheme to use and ability to encrypt/decrypt files in other directories.
